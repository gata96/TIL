여러가지 함수

1. CASE WHEN

```SQL
SELECT CASE WHEN 조건1 THEN 결과1
            WHEN 조건2 THEN 결과2
            ELSE 결과3
            END (AS NEW_컬럼명)
FROM 테이블이름;
```

```SQL
SELECT *,
        CASE WHEN USD > 10000 THEN 'BIG'
            WHEN USD > 5000 THEN 'MIDDLE'
            ELSE 'SMALL'
            END AS 그룹
FROM STOCK_INFO
ORDER BY USD DESC;
```
